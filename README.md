# Echo - Campus Lost and Found

Echo is a university-focused mobile app built to replace chaotic lost-and-found WhatsApp groups.
It provides a clean, private, and structured way for students to report lost items, post found items,
and coordinate returns safely inside the app.

## Idea
- Solve campus lost-and-found confusion with organized listings.
- Keep communication private using in-app chat instead of sharing phone numbers.
- Reduce feed clutter with clear status tracking (Lost, Found, Resolved).
- Improve recovery speed with searchable posts, categories, and quick access to details.

## Core Screens
1. Authentication: university user entry point.
2. Home Feed: browse all lost/found posts with filters.
3. Add Item: submit a lost or found report with image and description.
4. Item Detail: view full item info and actions.
5. Messages: private chat between owner and finder.
6. Profile/My Posts: manage active posts and mark items resolved.

## How It Works
1. User signs in and lands on the Home Feed.
2. Feed shows recent posts with status badges and location/time.
3. User can create a report from the Add Item form.
4. Tapping a post opens Item Detail for full context.
5. Contact action opens chat to coordinate handover.
6. Once returned, the post is marked resolved in Profile.

## Current Scope
- Frontend-first Flutter UI using GetX navigation and theming.
- Light and dark themes with a shared design system.
- Dummy data for lists, messages, and item details.

## Vision
Echo aims to become a trusted campus utility that makes returning items faster,
safer, and more reliable for every student.
